# soc-support-dashboard
To better assist the SOC Support with their duties, this is an internal dashboard for keeping track of commonly accessed tools and resources.


# Current Tooling

## Browser History Viewer

Upload and analyse a sqlite db file for potential malicious activity.

## Resource Dashboard

A "one-stop shop" for resources commonly used by the team. 

## Bulk Triage

An IOC extractor for log files and text dumps.

## SIEM Library

A list of common queries for using within SIEM.

## Network (In progress)

Use an API to look up details surrounding IP addresses, hashes and CVEs. Contains a variety of other handy tools that can help with analysis and working with partners.

## File and Cloud

Some simple queries and tools for analysing ioc's and data points.

## Cyberchef (refresh the window if it's scaled incorrectly) 

A built in browser in the browser for cyberchef

## Cheatsheets

A list of common cheatsheets to help with analysis and debugging of common security issues.
